California Renewable Curtailments Study¶

Foreword¶

The work done in this project was done in an academic context as an independent study. The original course proposal set out to define a domain-oriented problem to learn and apply statistical learning methods. Many thanks go out to Peter Alstone for his guidance, metorship and support.

The original course goals focused on developing a familiarity with the opensource ML and statistical learning ecosystem.
